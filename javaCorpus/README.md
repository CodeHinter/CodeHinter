# To Run Preprocess

> ```bash
> cd ./javaCorpus
> ```
> 
> ```shell
> pip install javalang
> ```
> 
> ```shell
> python preprocess.py input_file output_file
> ```



> This preprocess will convert Java source code into AST tree using javalang



> Run the preprocess file to the data set: train.txt, test.txt, dev.txt
> 
> The node is denoted with prefix "-"
> 
> The tree level is denoted with the number of space " " before the node
> 
> Each snippet of code is separated by an empty line
